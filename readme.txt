Hướng dẫn chạy project:
Yêu cầu: 
- Đã cài nodeJS
- Visual studio:
	+ Đã cài Entity Framework
	+ Đã cài Microsoft.AspNet.Cors
Các bước thực hiện: 
B1: Clone project từ github:
B2: Mở project API trên Visual studio
B3: Import file database.sql vào sql server và tạo database
B4: Xóa Model hiện tại và tạo mới Model Code First 
B5: Chỉnh sửa tên Entity ở dòng 11 trong các file DAO/
B6: Chỉnh sửa dòng 13 trong các file Controllers/ (nếu khác server/port)
B7: Khởi chạy project API
B8: Mở project NodeJS trong Visual studio code
B9: Mở terminal và gõ các lệnh sau:
	- npm i
	- npm start
B10: Vào trình duyệt, gõ localhost:3000 (thay đổi port tương ứng nếu có sửa đổi)
